In this folder, stl's are placed (made in sketchup). They can be read by openscad (and possibly converted to .scad later-on).
Some official 3D models for gridbeam have been made available, so sketchup is an obvious modelling program, to be used besides openscad and openjscad.
See https://3dwarehouse.sketchup.com/user/0096635206783583643122642/GridbeamSolutionscom?nav=collections
